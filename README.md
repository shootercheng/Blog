# Blog
简洁博客

## 项目启动步骤

### 下载 Ueditor

[Ueditor](https://ueditor.baidu.com/website/download.html)

选择 [1.4.3.2 Jsp 版本] UTF-8版

### 将 utf8-jsp目录下的jar包拷贝到WEB-INF/lib目录下

utf8-jsp\jsp\lib

### 配置 /static/ueditor/jsp/config.json

imagePathFormat
